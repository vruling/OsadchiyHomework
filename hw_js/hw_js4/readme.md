## Теоретический вопрос

1. Опишите своими словами, что такое метод обьекта

## Ответ 

1. Метод обьекта это свойство объекта, которое является функцией.
Метод определяется так же, как и обычная функция, за тем исключением, что они присваиваются свойству объекта.
Метод вызваем через точку например newUser.getLogin();
